# Changelog

All notable changes to Outcomist will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial MVP release
- Multi-project workspace with grid layout
- Four project types: game, trip, content, presentation
- Real-time AI chat with Claude
- Server-Sent Events (SSE) streaming
- Automatic file generation and updates
- Three views per project: Agent, Preview, Files
- Onboarding flow for new users
- SQLite database for persistence
- Docker deployment setup
- Comprehensive documentation

### Features
- Create and manage multiple projects
- Chat with specialized AI agents per project
- Stream AI responses in real-time
- Generate files automatically during conversation
- Preview generated content
- View all project files
- Persistent storage of projects, sessions, and messages

### API
- REST API for projects, sessions, messages, files
- SSE streaming for real-time updates
- Health check endpoint
- CORS configuration for development

### Documentation
- README with quick start guide
- Deployment guide with Docker and cloud options
- Complete user guide
- API reference documentation
- Contributing guidelines

## [0.1.0] - 2024-01-15

### Initial Release
- MVP version of Outcomist
- Basic multi-project functionality
- AI-powered project creation
- File generation and management
- Docker deployment ready
