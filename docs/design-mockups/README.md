# Design Mockups

This folder contains HTML mockups used during the design process for Outcomist.

## Files

- `multi_project_workspace.html` - Grid layout with multiple project cards
- `canvas_workspace.html` - Canvas-focused layout exploration
- `workspace_with_status.html` - Status indicator variations
- `workspace_with_view_toggle.html` - View mode switching concepts
- `workspace_agent_view.html` - Agent/chat view design
- `workspace_with_quick_actions.html` - Quick action buttons
- `dual_mode_workspace.html` - Split view concepts
- `project_view_comparison.html` - Side-by-side layout comparisons

## Purpose

These mockups were created to:
- Explore layout options before implementing
- Compare different UI approaches visually
- Get alignment on design direction
- Document design decisions

## Usage

Open any HTML file in a browser to see the static mockup. These are reference materials, not part of the production code.
